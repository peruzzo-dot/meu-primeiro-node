# 🚀 Meu Primeiro Servidor com Node.js e Express

Este projeto marca o início da minha jornada no desenvolvimento Back-end, focado em construir aplicações robustas, escaláveis e eficientes. Aqui, saí do ambiente do navegador para rodar JavaScript diretamente no servidor.

## 🛠️ Tecnologias Utilizadas

* **Node.js**: Ambiente de execução JavaScript server-side.
* **Express.js**: Framework rápido e minimalista para gerenciamento de rotas e servidores web.
* **Chalk**: Biblioteca para estilização de logs no terminal, facilitando o monitoramento do sistema.
* **Nodemon**: Ferramenta de automação que reinicia o servidor automaticamente a cada alteração no código, otimizando o fluxo de trabalho.

## 🎯 Aprendizados e Desafios

Durante o desenvolvimento deste projeto, explorei conceitos fundamentais de engenharia de software e infraestrutura:

* **Configuração de Ambiente**: Gestão de políticas de execução no Windows PowerShell e configuração do ecossistema NPM.
* **Arquitetura de Servidor**: Criação de um servidor HTTP capaz de escutar requisições em portas específicas (Porta 3000).
* **Roteamento Dinâmico**: Implementação de rotas que processam parâmetros da URL para entregar respostas personalizadas ao usuário.
* **Versionamento Profissional**: Uso de `.gitignore` para manter o repositório limpo e boas práticas de commits no Git/GitHub.
